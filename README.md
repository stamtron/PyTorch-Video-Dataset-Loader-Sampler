# pytorch-video-dataset-loader-sampler
PyTorch Video Dataloader for SpatioTemporal models, assuming your folder structure looks like this:  
  
root/  
    - action0/  
        -event0/  
            -frame00.png  
            -frame01.png  
            - ...  
        -event1/  
            -frame00.png  
            -frame01.png  
            - ...  
    - action1/  
        -event0/  
            -frame00.png  
            -frame01.png  
            - ...  
        -event1/  
            -frame00.png  
            -frame01.png  
            - ...
